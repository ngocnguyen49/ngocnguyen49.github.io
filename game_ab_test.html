A/B TESTING IN GAME RETENTION

Improving user engagement and retention through data-driven experimentation

PROJECT OVERVIEW

In this project, we designed an A/B testing experiment to evaluate different versions of a mobile game feature. The objective was to assess which version drives higher user retention and engagement.

EXPERIMENT DESIGN

  • Objective: Improve Day 1 and Day 7 retention
  • Variants: Version A (control) vs. Version B (new feature)
  • Metric: Retention rate, average session time, in-app purchases
  • Tool: Python (pandas, scipy), SQL, Power BI

RESULTS

Based on the results, Version B led to a statistically significant improvement in Day 7 retention by 8%. Additionally, session duration increased and in-app purchases rose slightly.
