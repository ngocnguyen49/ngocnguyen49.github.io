A/B TESTING IN GAME RETENTION

Improving user engagement and retention through data-driven experimentation

PROJECT OVERVIEW

In this project, we designed an A/B testing experiment to evaluate different versions of a mobile game feature. The objective was to assess which version drives higher user retention rate and win rate.

EXPERIMENT DESIGN

  • Objective: Retention rate, win rate of user after watching tutorial
  • Variants: Version 1.5.2 (control) vs. 1.6.0 (new feature)
  • Metric: Retention rate, win rate
  • Tool: Python (pandas, scipy)

RESULTS

Based on the results, The analysis revealed no statistically significant difference in retention rates between Version 1.5.2 and Version 1.6.0. However, a statistically significant difference was observed in win rates, indicating that Version 1.6.0 had a measurable impact on user performance.
